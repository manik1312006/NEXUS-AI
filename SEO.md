# NEXUS AI SEO Plan

Primary URL: https://nexusai.manikmehta.workers.dev/

## Positioning

NEXUS AI should rank as a free multi-agent AI chat platform for people who want one browser-based workspace for multiple AI models, prompt enhancement, web search, deep research, voice input, TTS, and chat history.

## On-Page Targets

Homepage:
- Title: NEXUS AI | Free Multi-Agent AI Chat Platform
- H1: NEXUS AI / Multi-Agent Chat
- Meta description: Focus on free multi-agent AI chat, prompt enhancement, web search, deep research, voice input, and chat history.
- Search intent: Users looking for an all-in-one AI chat hub.
- Internal conversion goal: Google sign-in or guest login.

Chat page:
- Status: noindex, follow.
- Reason: It is an authenticated app page with thin public content and redirects unauthenticated users.

## Technical SEO Checklist

Completed in this repo:
- Homepage title, description, canonical, robots meta, Open Graph, Twitter Card, and structured data.
- SoftwareApplication, WebSite, and Person JSON-LD on the homepage.
- `robots.txt` and `sitemap.xml`.
- `site.webmanifest`.
- Netlify `_headers` for sitemap/robots content types, logo caching, and chat noindex header.
- Backend API noindex header and backend robots disallow.
- Local logo references instead of loading the logo through raw.githubusercontent.com.

After deployment:
- Confirm `https://nexusai.manikmehta.workers.dev/robots.txt` is accessible.
- Confirm `https://nexusai.manikmehta.workers.dev/sitemap.xml` is accessible.
- Submit the sitemap in Google Search Console.
- Inspect the homepage URL in Search Console and request indexing.
- Use Rich Results Test or Schema Markup Validator to confirm JSON-LD is readable.
- Run Lighthouse for mobile performance and accessibility.

## Backlink Plan

Good backlink targets:
- GitHub repo README and project topics.
- Personal portfolio page for Manik Mehta.
- Product Hunt launch page.
- Indie Hackers project page.
- Dev.to or Hashnode tutorial about building a multi-agent AI chat app.
- Medium article explaining the prompt enhancer or deep research engine.
- AI tool directories that manually review products.
- Student developer communities and college project showcases.
- Relevant Reddit/Discord communities only when sharing is allowed and genuinely useful.
- YouTube demo with homepage and GitHub links in the description.

Avoid:
- Paid links that pass ranking credit.
- Link farms, comment spam, profile spam, and automated directory blasts.
- Large-scale guest-post exchanges.
- Exact-match anchor text repeated everywhere.
